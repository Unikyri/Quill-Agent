# Audit &amp; access logs

> **Source:** https://docs.qwencloud.com/developer-guides/security-compliance/audit-logs

Track API usage for compliance

 Copy page Qwen Cloud provides usage tracking through the Analytics dashboard. These tools help you monitor API consumption, investigate anomalies, and support compliance reporting.
## [​ ](#analytics-dashboard) Analytics dashboard

The [Analytics](https://home.qwencloud.com/analytics) page shows aggregated usage metrics for your workspace:

- **Token consumption**: Total input and output tokens over time.

- **Request counts**: Number of API calls by model and time period.

- **Latency**: Average response times.

- **Success rate**: Percentage of successful vs. failed requests.


You can filter by time range and model to isolate specific patterns. The page also provides per-model operational metrics, including rate limit tracking and error rates.
The [Logs](https://home.qwencloud.com/analytics/request-logs) tab provides request-level audit trails with per-request token usage, latency, and status codes. Logs are retained for 14 days. For details, see [MLOps & observability](/developer-guides/integrations/mlops-observability#request-logs).
