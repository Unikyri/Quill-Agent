 Copy page [​ ](#may-15-2026) May 15, 2026 ## Model production
The console now supports [model production](https://home.qwencloud.com/model-production/overview) workflows, including dataset management, fine-tuning (SFT/LoRA), and model deployment. You can upload training data, create fine-tuning jobs, and deploy trained models as inference services.Also launched [Batch API](https://home.qwencloud.com/model-production/batch-api) in the console for submitting and managing batch inference jobs.
